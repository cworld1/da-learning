13.3. The Poisson Distribution#

The Poisson is a distribution that takes values on the non-negative { 0,1,2,3,... } integers and it has no upper limit. æ³Šæ¾æ˜¯ä¸€ç§åœ¨éè´Ÿ{0,1,2,3ï¼Œâ€¦}æ•´æ•°ä¸Šå–å€¼çš„åˆ†å¸ƒï¼Œå®ƒæ²¡æœ‰ä¸Šé™ã€‚
