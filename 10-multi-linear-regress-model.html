Relationship between birth weight and gestational age...

For gestation \(\leq 294\) days weâ€™ll use the familiar simple linear regression model

\[ \text{E[bwt]}=\beta_0+\text{gestation}\times\beta_1 \]

Weâ€™d like to extend this model by adding an extra term so that the slope changes when gestation \(> 294\). That is,

\[ \text{E[bwt]}=\beta_0+\text{gestation}\times\beta_1+\text{v}\times\beta_2 \]

where v is some suitable explanatory variable. What should v be?

  • For gestation \(\leq 294\) the extended model is just the simple linear regression model, so that means v = 0 when gestation â‰¤ 294.

  • For gestation \(> 294\) we need another slope effect for gestational age. In fact, we need \(v =\) gestation \(- 294\).

Letâ€™s create the new explanatory v = 294 that is described gestation above. Weâ€™ll give it the name because it is the number of days ODdays that the baby is overdue.

Babies.df$ODdays <- ifelse(
    Babies.df$gestation < 294,
    0,
    Babies.df$gestation - 294
