15.1. Binary (Bernoulli) data, odds and log-odds#

Here we are considering the situation where the response can only take two possible values. These might be coded in the form of:

  • Zeros or ones.

  • TRUE or FALSE.

  • Yes or No.

  • Success or Failure.

  • Or any other pair of categorical values.

Bernoulli random variables(ä¼¯åŠªåˆ©éšæœºå˜é‡):

If Y is a Bernoulli random variable with parameter p, then Y will take the value 1 with probability p, and the value 0 with probability 1 - p. Since it is a probability, p must be a value that is between 0 and 1, i.e. \(p âˆˆ [0, 1]\).

15.2. Modelling log-odds#

Why log-odds?

\[ \text{Log-Odds}=\beta_0+\beta_1x \]

Log-Odds can be any real number.
