The standard notation for a quadratic curve is:

\[ y = a x^2 + b x + c \]

Here we will use different notation: \(\beta_0 = c\), \(\beta_1 = b\) and \(\beta_2 = a\) and use the quadratic curve to describe the expected value of our dependent variable \(y\). That is, we will use the following notation:

\[ E[Y|x] = \beta_0 + \beta_1 x + \beta_2 x^2 \]

If \(\beta_2 > 0\), then the quadratic has slope that increases with increasing x(æ–œç‡éšç€ x å¢å¤§è€Œå¢å¤§). If \(\beta_2 < 0\), then the quadratic has slope that decreases with increasing x. If \(\beta_2 = 0\), then the quadratic(è¯¥â€œäºŒæ¬¡æ›²çº¿â€) has a constant slope(å€¾æ–œç›´çº¿çš„å¤–è§‚).
